# fpc-llama.cpp

Freepascal bindings for Llama.cpp

Derived from https://github.com/Kagamma/llama-pas, but updated to work with llama.cpp, commit 324f3403d54ae4499a1d68623161015f7419fb76 Thu Sep 21 21:08:20 2023 

Also provided is llama-cli, an example application to do basic inference of gguf models.

Tested with fpc 3.2.2, Lazarus 3.0.0.
